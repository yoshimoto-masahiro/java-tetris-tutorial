# Java Tetris Tutorial 🎮

Java初心者向けに、Swingで作る簡単なテトリスゲームのチュートリアルです。

## 特徴
- Java標準ライブラリ（Swing）だけで動く
- キーボード操作：← → ↓ 回転（↑）
- ファイル3つだけのシンプル構成

## クラス構成
- `Tetris.java`：起動用のメインクラス
- `Board.java`：ゲームロジック＆描画
- `Shape.java`：テトリミノの定義と回転処理

## 実行方法
1. 任意のJava IDEでプロジェクトを作成
2. 上記3ファイルをコピー＆ペースト
3. `Tetris.java` を実行！
